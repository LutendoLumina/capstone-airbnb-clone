# Airbnb Admin Dashboard Clone — Frontend

This project is an Admin Dashboard for an Airbnb clone built using React.

## Features Completed
* **Top Header Navigation:** Includes the logo placeholder, an interactive search button, and a profile menu toggle that shifts dynamically based on user session status.
* **Inline Login Form Validation:** Captures email structure and checks for password thresholds before routing to the workspace panel via dummy credentials (`admin@airbnb.com` / `password123`).
* **Create Listing Form Layout:** A comprehensive form panel containing vanilla HTML inputs capturing titles, prices, descriptions, metrics (bedrooms, bathrooms, max guests), separate extra pricing lines (cleaning fees, service fees, occupancy taxes, weekly discounts), and checked amenities options.

## Tech Stack Installed
* **Frontend Library:** React.js
* **Icon Packs:** `react-icons` (FontAwesome, Ionicons)
* **Styling approach:** Pure Vanilla CSS
